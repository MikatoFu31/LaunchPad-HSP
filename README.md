# LaunchPad-HSP
LaunchPad値段高すぎるのでシミュレータを自作。
## 操作方法
12345678   
QWERTYUI   
ASDFGHJK   
ZXCVBNM,   
のそれぞれのキーで対応するパッドを押す。

キーボードで上4段を操作対象にするには「-」キーを、下4段を操作対象にするには「@」キーを押すと画面左上の「Switch」の値が変わる。
## 音楽の読み込み方法
[行番号]-[列番号].wav というファイルをソースファイルと同じディレクトリに保存すると起動時に自動的に読み込む。
例えば、1-1.wav は左上のボタンを押すと再生され、1-8.wavは右上のボタンを押すと再生される。
